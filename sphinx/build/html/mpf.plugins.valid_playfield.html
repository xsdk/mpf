mpf.plugins.valid_playfield module

MPF plugin that handles valid playfield tracking for a pinball machine.

class mpf.plugins.valid_playfield.ValidPlayfield(machine)

Bases: object

Base class for a valid playfield checking module.

Parameters:machine (MachineController) – A reference to the instance of the MachineController object.

Handler for the ball_drain relay event which is called when a ball drains when the playfield is not valid.

Sets up the valid playfield checking.

live_hit()

Called whenever a switch couting towards valid playfield is hit.

remove()

Cancels and removews the valid playfield verification checking.
